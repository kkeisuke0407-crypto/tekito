銀行からの借金を返せない時の対応を調べている方は、銀行カードローンやフリーローン、住宅ローン以外の借入返済に悩んでいるかもしれません。銀行からの借金も、滞納すれば督促や一括請求、保証会社からの請求に進むことがあります。この記事では、返せない時の流れと相談前の確認点をまとめます。

## 銀行からの借金を返せない時に起こること

銀行からの借金を返せない場合、まず督促や引き落とし不能の連絡が来ます。

滞納が続くと、一括請求や保証会社による代位弁済に進むことがあります。代位弁済とは、保証会社が銀行へ返済し、その後は保証会社から請求を受ける流れです。

銀行から借りているから安心というわけではなく、滞納が続けば法的手続きに進む可能性があります。

## 銀行借入で確認すること

相談前に、次の内容を確認しましょう。

| 確認項目 | 内容 |
|---|---|
| 借入種類 | カードローン、フリーローンなど |
| 残高 | いくら残っているか |
| 滞納期間 | 何回遅れているか |
| 保証会社 | どこが保証しているか |
| 給与口座 | 同じ銀行を使っているか |

給与口座と借入先の銀行が同じ場合、引き落としや口座管理が不安になる方もいます。相談時に伝えましょう。

## 放置した場合の流れ

銀行借入を放置すると、次のように進むことがあります。

| 段階 | 内容 |
|---|---|
| 滞納 | 電話や書面で督促 |
| 一括請求 | 残額をまとめて請求 |
| 代位弁済 | 保証会社が銀行へ支払い |
| 保証会社から請求 | 請求元が変わる |
| 法的手続き | 支払督促や訴訟の可能性 |

請求元が銀行から保証会社へ変わると、戸惑う方も多いです。書類を捨てずに保管しましょう。

## 返せない時の選択肢

| 状況 | 検討する方法 |
|---|---|
| 毎月返済できる余地がある | 任意整理 |
| 複数社の返済が重い | 任意整理や個人再生 |
| 返済見込みがない | 自己破産 |
| 住宅ローンもある | 個人再生を含めて相談 |

銀行カードローンも任意整理の対象になることがあります。ただし、同じ銀行の口座や保証会社への影響を確認する必要があります。

## 相談先に伝えるべきこと

銀行からの借金を相談する時は、次を伝えてください。

- 銀行名
- 保証会社名
- 借入残高
- 滞納期間
- 給与口座との関係
- 住宅ローンの有無
- 他社借入の有無

銀行借入だけでなく、カードや消費者金融もある場合は、全体で整理した方が判断しやすいです。

## FAQ

### 銀行カードローンも債務整理できますか？

可能性はあります。保証会社や口座への影響も含めて確認しましょう。

### 保証会社から請求が来たらどうすればいいですか？

代位弁済後の請求の可能性があります。書類を保管して相談しましょう。

### 給与口座が同じ銀行だと危険ですか？

状況によります。相談時に同じ銀行を給与口座にしていることを伝えてください。

### 銀行の借金だけ任意整理できますか？

可能な場合がありますが、他社借入も含めて全体を見た方がよいです。
