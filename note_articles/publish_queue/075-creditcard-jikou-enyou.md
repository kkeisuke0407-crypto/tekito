## クレジットカードの時効援用の基本

クレジットカードの未払いは、一定期間が経過すると消滅時効を主張できる可能性があります。

ただし、時効期間が過ぎただけで自動的に請求がなくなるわけではありません。時効援用の意思表示が必要です。

| 確認項目 | 見る理由 |
|---|---|
| 最終返済日 | 時効期間の起点を確認 |
| 裁判歴 | 判決があると期間が変わることがある |
| 債権回収会社 | 請求元が変わっている場合がある |
| 最近の支払い | 時効に影響する可能性 |
| 電話での発言 | 支払い約束に注意 |

## 古い請求でも放置は危険

何年も前のカード代金だからといって、督促状を捨てて放置するのは危険です。

裁判所から書類が届いた場合、期限内に対応しないと不利な結果になる可能性があります。通常の督促状と裁判所の書類は重みが違うため、封筒や差出人を確認しましょう。

## 債権回収会社から届くこともある

クレジットカード会社の未払いは、債権回収会社や弁護士法人から請求されることがあります。

名前を知らない会社からの通知でも、元のカード会社の債権を引き継いでいる場合があります。怪しいと決めつけず、請求元と元債権者を確認しましょう。

## 連絡や支払いの前に確認する

時効の可能性がある借金では、慌てて電話をしたり、一部だけ支払ったりしないことが大切です。

「分割で払います」「少し待ってください」といった発言が、支払いを認めたと見られる可能性があります。まずは資料を持って相談しましょう。

## 時効援用の流れ

一般的な流れは次の通りです。

1. 督促状や契約情報を確認する
2. 最終返済日や裁判歴を確認する
3. 時効の可能性を専門家に相談する
4. 内容証明などで時効援用通知を送る
5. 請求が止まるか確認する

自分で通知を出す場合も、内容や送付先の確認が重要です。
