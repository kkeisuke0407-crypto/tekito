## 任意整理中の住宅ローンは難しい？

任意整理中に住宅ローンを組むのは、一般的にはかなり難しいと考えた方がよいです。

任意整理中は、借金を分割返済している途中です。金融機関から見ると、すでに返済負担があり、信用情報にも影響がある状態です。そのため、住宅ローンのような大きな借入では不利になりやすいです。

まずは任意整理の返済を安定して続けることが優先です。

## 審査で見られやすいポイント

| 項目 | 内容 |
|---|---|
| 信用情報 | 任意整理や延滞の登録状況 |
| 返済中の残高 | 任意整理の残債 |
| 年収 | 住宅ローン返済能力 |
| 勤続年数 | 収入の安定性 |
| 他の借入 | 車ローンやスマホ分割 |
| 頭金 | 借入額を抑えられるか |

任意整理中に無理な住宅ローンを組むと、家計が再び苦しくなる可能性があります。

## 住宅購入を考える前にやること

住宅ローンを考えるなら、次の順番で準備しましょう。

1. 任意整理の返済を遅れず続ける
2. 完済予定を確認する
3. 信用情報を確認する
4. 頭金を貯める
5. 他の借入を増やさない
6. 家計の返済可能額を把握する

任意整理中に焦って申し込むより、完済後の準備を進める方が現実的です。

## 配偶者名義なら可能？

配偶者単独の収入と信用情報で審査する場合、可能性がないとは言えません。

ただし、収入合算、ペアローン、連帯保証を使う場合は、自分の信用情報が関係することがあります。夫婦で住宅購入を考えている場合は、事前に金融機関や専門家へ確認しましょう。
