## 個人再生の弁護士費用が払えないときの考え方

個人再生は、裁判所を通して借金の返済計画を立て直す手続きです。

任意整理より手続きが重く、弁護士費用も高くなりやすい傾向があります。ただし、費用がすぐ払えないからといって、相談をあきらめる必要はありません。

| 確認したいこと | 理由 |
|---|---|
| 分割払いの可否 | 初期負担を抑えられる場合がある |
| 着手金の時期 | いつ支払いが始まるか |
| 住宅ローンの有無 | 費用が変わることがある |
| 法テラス利用 | 収入条件に合えば立替制度を検討できる |
| 実費の範囲 | 裁判所費用なども確認 |

## 分割払いに対応する事務所を探す

債務整理を扱う事務所では、弁護士費用や司法書士費用の分割払いに対応している場合があります。

相談時には、総額だけでなく「毎月いくらなら払えるか」を正直に伝えることが大切です。

無理な支払い計画で契約すると、個人再生の返済計画そのものにも影響します。家計表を作り、生活費を引いた後に残る金額を確認しましょう。

## 法テラスを使える可能性もある

収入や資産が一定以下の場合、法テラスの民事法律扶助を利用できる可能性があります。

利用できれば、弁護士費用などを立て替えてもらい、後から分割で返済する形になります。

ただし、利用には審査があり、すべての事務所で同じように使えるわけではありません。法テラス対応の可否も相談先選びのポイントです。

## 相談前に整理する家計情報

費用が払えない場合ほど、家計の見える化が重要です。

- 毎月の手取り収入
- 家賃、住宅ローン
- 食費、通信費、保険料
- 借金の返済額
- 滞納している支払い
- 家族構成

個人再生は、手続き後も一定期間返済を続ける制度です。費用だけでなく、再生計画の返済が続けられるかも見てもらいましょう。

## 費用だけで相談先を決めない

費用が安いことは大切ですが、個人再生は住宅ローン、車、保証人、勤務先への影響など確認点が多い手続きです。

費用の安さだけで決めるより、説明がわかりやすいか、分割条件が現実的か、個人再生の実務に慣れているかを比較しましょう。
