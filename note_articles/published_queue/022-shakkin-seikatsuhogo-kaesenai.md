## 生活保護中に借金を返せない時の基本

生活保護を受けている状態で借金を返せない場合、無理に返済を続けるより、早めに相談した方がよいです。

生活保護費は、最低限度の生活を維持するためのものです。そこから借金返済を続けると、家賃、食費、医療、通信費などに影響が出る可能性があります。

返済が難しいなら、債務整理や自己破産を含めて相談することになります。

## 生活保護中に注意したいこと

借金がある状態で生活保護を受けている場合、次の点に注意が必要です。

| 注意点 | 内容 |
|---|---|
| 借金返済 | 生活費を圧迫しやすい |
| 新たな借入 | 収入や資産の扱いで問題になる可能性 |
| 督促 | 精神的負担が大きい |
| 裁判所書類 | 放置すると差し押さえなどの問題に進む可能性 |
| ケースワーカー | 状況によって相談した方がよい場合がある |

生活保護中は、借金問題だけでなく生活全体への影響を考える必要があります。

## 考えられる解決方法

生活保護中で返済原資がない場合、任意整理よりも自己破産を含めて検討することが多くなります。

| 方法 | 向きやすい状況 |
|---|---|
| 任意整理 | 毎月返済できる余裕がある |
| 自己破産 | 返済の見込みが立たない |
| 時効援用 | 長期間返済していない古い借金 |
| 法テラス相談 | 費用が不安な場合 |

ただし、どの方法が合うかは借金額、財産、過去の返済状況で変わります。自己判断だけで進めるのは避けた方が安全です。
