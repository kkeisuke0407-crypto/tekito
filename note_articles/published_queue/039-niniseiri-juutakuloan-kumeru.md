## 任意整理後に住宅ローンを組めるか可能性

任意整理後でも、将来的に住宅ローンを組める可能性はあります。

ただし、任意整理直後は信用情報に事故情報が登録されるため、住宅ローン審査に通りにくいと考えた方がよいです。登録期間が過ぎた後でも、収入、勤続年数、頭金、他の借入、返済履歴などを総合的に見られます。

「任意整理したら一生住宅ローンが組めない」わけではありませんが、時間と準備が必要です。

## 審査で見られやすいポイント

| 項目 | 見られる理由 |
|---|---|
| 信用情報 | 過去の延滞や債務整理 |
| 年収 | 返済能力 |
| 勤続年数 | 収入の安定性 |
| 他社借入 | 返済負担率 |
| 頭金 | 借入額を抑えられるか |
| 返済履歴 | 家計管理の安定性 |

任意整理後は、まず現在の返済を遅れず続け、完済後も家計を安定させることが大切です。

## 住宅ローン前に確認すること

住宅ローンを考える前に、次を確認しましょう。

- 任意整理の完済時期
- 信用情報の登録状況
- 現在の借入残高
- 携帯分割やカード利用の有無
- 頭金を用意できるか
- 返済負担率に無理がないか

信用情報は本人開示で確認できます。ローン申込前に自分の状況を把握しておくと、無駄な申込を減らしやすくなります。

## 焦って申し込まない方がいい理由

任意整理直後に住宅ローンへ申し込んでも、審査に通らない可能性が高いです。

短期間に何度も申し込むと、申込履歴が残ることがあります。まずは信用情報、家計、頭金を整えてから検討した方が現実的です。
