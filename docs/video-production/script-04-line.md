## スライド仕様

### スライド1（0-3秒）iPhoneロック画面
- **背景:** ロック画面の写真風（暗めの夜空、グラデーション）
- **時計表示:** 23:47（夜遅い時間を演出）
- **日付:** 日曜日
- **LINE通知バナー:**
  ```
  LINE  いま
  【店長】明日 朝5時 入って
  ```
  - フォント: SF Pro Display Bold（iOS風）
  - 通知背景: 白半透明
- **上部テロップ:** 「あなたのスマホ、こうなってない?」
  - 色: #FFFFFF
  - サイズ: 60pt
  - ストローク: 黒3px

### スライド2（3-8秒）LINEチャット画面
- **背景:** #7090A8（LINE標準背景・夜モード）
- **ヘッダー:**
  - アイコン: 弁護士アイコン（スーツ姿のシルエット）
  - 名前: 「弁護士による退職代行」
  - サブ: 「⚖ 認証済み」マーク
- **ユーザー側吹き出し（右・緑）:**
  - テキスト: 「もう限界です…でも自分では言えません」
  - 色: #06C755背景 / 白文字
- **タイピング中アニメーション（5秒-5.5秒）:**
  - 「・・・」が動く
- **弁護士側吹き出し（左・白）:**
  - テキスト: 「大丈夫です。明日の朝、こちらから連絡します」
  - 色: 白背景 / 黒文字

### スライド3（8-13秒）翌朝の画面
- **時計表示:** 9:01
- **日付:** 月曜日
- **店長からの通知（暗くフェード）:**

- **オーバーレイテロップ（中央）:**
  - 「翌朝9:01」（上部）

## CapCut組み立て手順

1. **LINE画面モック作成（事前準備）**
   - 推奨ツール: **Fake Chat Maker** (https://fakedetail.com/) または **LINE風スクショ作成アプリ**
   - ユーザー名: 「弁護士による退職代行」（ガイア名NG）
   - 3画面分書き出し（PNG）

2. **iPhoneロック画面モック**
   - 推奨: Canvaの「iPhone Lock Screen Mockup」テンプレ
   - 通知バナーは別レイヤーで重ねる

3. **CapCut編集**
   - 各画面を3-5秒ずつ並べる
   - ズームエフェクト（軽く）で動きを出す

4. **SE追加**
   - LINE通知音「ピコン」: 0秒
   - タイピング音: 5.0-5.5秒
   - 通知音: 8.5秒（翌朝の通知）

5. **ナレーション追加**
   - VOICEVOX書き出しwav 2本配置

6. **BGM**
   - 8秒以降のみ。それ以前は無音 or SEのみで緊張感を演出
   - Lo-fi 25%音量

7. **字幕**: `script-04-line.srt`をインポート

8. **書き出し**: MP4 / 1080P / 30fps
