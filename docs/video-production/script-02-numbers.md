## CapCut組み立て手順

1. **新規プロジェクト**: 1080×1920 / 30fps / 15秒

2. **スライド作成**: Canvaで4枚PNG書き出し（推奨）
   - ※テキスト数字が多いため、Canva使う方がCapCut直編集より速い

3. **音声追加**:
   - VOICEVOXでセリフ①②をwav書き出し
   - セリフ①を3:00、セリフ②を8:00に配置

4. **BGM**:
   - 推奨: TikTok Creative Center → 「Lo-fi」「Chill」検索
   - 音量: 25%
   - 0-1秒は無音、1秒以降フェードイン

5. **字幕**: `script-02-numbers.srt`をインポート

6. **書き出し**: MP4 / 1080P / 30fps
