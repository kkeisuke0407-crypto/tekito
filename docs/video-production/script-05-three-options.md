## スライド仕様

### スライド1（0-3秒）新聞風
- **背景:** #F5F1E8（古紙色）
- **新聞ヘッダー風レイアウト:**
  - 上部に「号外」風タイトルバー（赤）
  - 大見出し: 「退職代行業者 弁護士法違反で逮捕」
    - フォント: 明朝体 Bold（源ノ明朝 or Noto Serif JP）
    - 色: #0A0A0A
    - サイズ: 75pt
  - 日付: 「2026年2月」（サイズ40pt、左上）
- **下部テロップ（オーバーレイ）:**
  - 「もう、業者選びを間違えられない」
  - サイズ: 55pt
  - 色: #C8102E（赤）

### スライド2（3-8秒）3つの選択肢
- **背景:** #FFFFFF
- **タイトル:** 「あなたが選べるのは、3つ」（サイズ50pt、上部）
- **3つのボックス（縦並び）:**
  ```
  ┌─────────────────────────┐
  │ ① 民間業者              │ 背景: #BDBDBD（グレー）
  │   非弁リスクあり        │ 色: 白
  └─────────────────────────┘
  ┌─────────────────────────┐
  │ ② 労働組合              │ 背景: #FBC02D（黄）
  │   交渉◯ / 訴訟✕         │ 色: #0A0A0A
  └─────────────────────────┘
  ┌─────────────────────────┐
  │ ③ 弁護士                │ 背景: #2E7D6A（深緑）
  │   全部 ◯                │ 色: 白
  └─────────────────────────┘
  ```
- **アニメーション:** ①②③が1秒間隔でスライドイン

### スライド3（8-13秒）弁護士の優位性
- **背景:** #2E7D6A（深緑）
- **タイトル:** 「弁護士だからできること」（白、サイズ65pt）
- **リスト（1秒間隔で出現）:**
  ```
  ◯ 有給消化の交渉
  ◯ 残業代の請求
  ◯ 損害賠償への対応
  ◯ 公務員・自衛隊対応
  ```
  - 色: 白
  - サイズ: 55pt
  - ◯の色: #FFEB3B（黄）

### スライド4（13-15秒）CTA
- **背景:** 白
- **テキスト:**
  - 「LINE無料相談」（緑、サイズ70pt）
  - 「→ コメント欄のリンクから」（黒、サイズ50pt）
  - 「相談だけでもOK」（グレー、サイズ40pt、下部）

## CapCut組み立て手順

1. **スライド作成**: Canvaで4枚PNG書き出し

2. **ナレーション**: VOICEVOXでセリフ①②を書き出し
   - セリフ①: 3:00配置
   - セリフ②: 8:00配置

3. **BGM/SE**:
   - 0-1秒: 無音
   - 1-3秒: 重低音SE（ニュース風）
   - 3-8秒: 重低音BGM継続（低音量）
   - 8-15秒: Lo-fi（明るめ）にフェード

4. **字幕**: `script-05-three-options.srt`をインポート

5. **書き出し**: MP4 / 1080P / 30fps
