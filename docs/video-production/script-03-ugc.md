## 演者への指示書（撮影台本）

### 場所
- 自宅の部屋風（生活感あるけど散らかってない）
- 自然光が入る場所
- 背景: ベッド / カーテン / シンプルな壁

### 衣装
- パーカー or Tシャツ（カジュアル）
- メイク薄め
- 髪はおろし or ラフお団子

### スマホ
- 縦持ち、目線の高さで自撮り
- 手ブレOK（むしろリアル感が出る）

### 演技指示

**0-3秒（つかみ）:**
- カメラ目線で前のめり
- 「マジでこれ、私が辞めた時の話聞いて」を早口気味
- 表情: ちょっと興奮した感じ

**3-8秒（過去の辛さ）:**
- 目線を少し下に
- 「辞めますって言ったら…」を一呼吸置きながら
- 「お前のせいで人手不足だ」は怒鳴られた時の声色を再現
- 涙目になれたらベスト（ならなくてもOK）

**8-13秒（救われた話）:**
- 表情を一気に明るく
- 「友達に教えてもらった『弁護士の退職代行』」を強調
- 安堵の笑顔で「辞められたんだよね」

**13-15秒（CTA）:**
- 軽く笑いながら「気になる人はプロフのリンク」
- 指で画面下を指す動作

## CapCut組み立て手順

1. **撮影素材を取り込む**
   - 演者撮影のmp4をインポート

2. **不要部分カット**
   - 各シーンの間延びを削除
   - 全体で15秒に収める

3. **テロップ追加**
   - `script-03-ugc.srt`をインポート
   - 字幕スタイル: 太字+黒ストローク

4. **BGM**
   - TikTok Creative Center → 「emotional」「感動」「Lo-fi sad」
   - 音量: 15-20%（演者の声を邪魔しない）

5. **エフェクト**
   - 3-8秒: 軽くビネット（暗め）
   - 8-13秒: 明度+10%、彩度+15%（明るく見せる）

6. **書き出し**: MP4 / 1080P / 30fps
