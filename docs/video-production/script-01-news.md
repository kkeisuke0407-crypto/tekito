## VOICEVOX用ナレーション原稿

**推奨スピーカー:** 玄野武宏 / ノーマル（男性アナウンサー風）
**速度:** 1.05倍
**音量:** 1.0
**抑揚:** 1.1（やや強め）

### セリフ①（0-3秒・約2.8秒）
```
にせんにじゅうろくねん にがつ、たいしょくだいこうぎょうかいの さいだいてのしゃちょうが、
べんごしほういはんの うたがいで たいほされました。
```
※VOICEVOXに貼る時は通常表記でOK:
```
2026年2月、退職代行業界の最大手の社長が、弁護士法違反の疑いで逮捕されました。
```
※「最大手」が「さいだいで」と誤読されたら、辞書登録で「さいだいて」に修正

### セリフ②（3-8秒・約4.8秒）
```
弁護士資格のない業者が、会社と交渉したり、弁護士を紹介して手数料を取ると、違法になります。
```
※「業者」は「ぎょうしゃ」読み確認
※「手数料」は「てすうりょう」

### セリフ③④（8-15秒）
**ナレーションなし**（テロップとBGMのみで間を作る）

## CapCut組み立て手順

1. **新規プロジェクト作成**
   - 解像度: 1080×1920（縦・9:16）
   - フレームレート: 30fps
   - 長さ: 15秒

2. **スライド作成**
   - 各スライドは画像書き出し（Canva / Figmaで作成 → PNG）
   - もしくはCapCutの「テキスト」機能で直接作成

3. **音声追加**
   - VOICEVOXで音声書き出し（wav） → CapCutにインポート
   - セリフ①を0:00に、セリフ②を3:00に配置
   - 8秒以降はナレーション無音

4. **BGM追加**
   - 0-3秒: ニュース速報音SE（CapCut素材ライブラリ「速報」「ニュース」で検索）
   - 3-8秒: 重低音BGM（「サスペンス」「警告」で検索）
   - 8-15秒: 落ち着いたBGM（「ピアノ」「希望」で検索）
   - 音量: BGM 20-30%、ナレーション 100%

5. **テロップ字幕**
   - `script-01-news.srt` をCapCutにインポート
   - 「字幕」→「字幕を追加」→「ローカル字幕をインポート」

6. **書き出し**
   - 解像度: 1080P
   - フレームレート: 30fps
   - 形式: MP4
