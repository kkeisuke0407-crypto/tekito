# テンプレ③：「不倫×世界・歴史・社会」雑学型

## 実績（伸びている新ジャンル）
- 不倫が世界史を変えたランキング（10万超え）
- 不倫に寛容な国・厳しい国ランキング（10万超え）
- 不倫で払った違約金ランキング（10万超え）
- 戦国・江戸時代は不倫＝死罪だった

## なぜ伸びる
当事者だけでなく「興味本位で見る層」を取り込めるためリーチが拡大する。
**今後最も伸びしろがあるジャンル**（HANDOFF.md §4-3）。

## 使い方
1. 「歴史」「世界」「統計」「文化比較」「芸能・著名人（報道ベース）」が軸
2. 数字・固有名詞・地名で具体性を上げる
3. 実名公人を扱うときは**運営者の許可必須**、性加害疑惑との混在NG（HANDOFF.md §8-6）
4. 順位と国名・人名の整合性チェックを徹底（過去ミスあり HANDOFF.md §8-4）
