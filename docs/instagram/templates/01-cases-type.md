# テンプレ①：「同じ〇〇でも変わるN個のケース」型

## 実績
- 32.3万再生（慰謝料が跳ね上がる6つのケース 第1弾）
- 20万再生（慰謝料が変わるケース 第3弾）※後に1万との訂正あり
- **同じテーマ3連続は急落リスク**。2〜3本他テーマを挟む。

## 使い方
1. テーマ：慰謝料・財産分与・養育費・公正証書など、金額やルールが個別事情で変わるもの
2. ケース数：5〜7個が黄金。多すぎると失速、少なすぎると物足りない
3. 各ケースは「具体的な数字差」「判例ベース」を含めると説得力↑
4. **過去弾との被りチェック必須**（HANDOFF.md §8-2 参照）

---

## 台本テンプレ

```
# 同じ[テーマ]でも[結果]が[N倍/円差]変わる[N個]のケース

- フォーマット: ①ケース型
- カテゴリ: [慰謝料/財産分与/etc.]
- 関連弾: 第N弾
- 作成日: YYYY-MM-DD

## 1. フック
「同じ不倫でも慰謝料が[N倍]違う、その理由を判例から[N個]紹介します」

## 2. ボディ（各ケース）
1. **[ケース名]** — [基本相場 → 実際の金額] / 理由（判例ロジックを1行）
2. **[ケース名]** — [基本相場 → 実際の金額] / 理由
3. ...

## 3. オチ／CTA
「同じ不倫でも、状況次第でこれだけ変わります。」
「@lawyer.nakanohito 詳しくはプロフのリンクへ」

## 4. キャプション
[投稿欄に貼る本文、絵文字なし]
```
