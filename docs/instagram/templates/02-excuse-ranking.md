# テンプレ②：「裁判で出た言い訳」ランキング型

## 実績
- 32.3万再生（第1弾）→ 20万再生（第2弾）と安定して高い
- **必ず過去弾の全リストを照合してから新規調査**（HANDOFF.md §8 ／被り事故が複数回）

## 使い方
1. ベスト5形式が王道
2. 「実際に裁判で主張された」感を強調すると当事者性が出る
3. 言い訳の中身は「現実離れしてるけど真顔で主張された」みたいなギャップが効く
4. **被りチェックは必須中の必須**
