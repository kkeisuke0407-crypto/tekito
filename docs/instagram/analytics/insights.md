| 指標 | 現状 | 業界平均 | 課題 |

| 総ビュー（30日） | 266万 | - | リーチは伸び続けている |

| 外部リンクタップ | 9件 / 126万ビュー時点 | - | サイト導線が機能していない |

### 原因仮説
- アカウントの性質上「悩みが解決すると離脱する」構造的問題
- フォロワーになる動機が弱い（情報を都度消費して終わる）
- プロフィールリンクへの誘導文言が弱い／タイミングが悪い

### 取った対応
- フォロワー数より「リーチを活かして毎回バズらせてサイトに流す」戦略に転換
- キャプション末尾を「@lawyer.nakanohito 詳しくはプロフのリンクへ」に統一
- 直接的なフォロー誘導文言は削除（信頼感重視）
