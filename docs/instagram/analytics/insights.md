- `undefined is not an object (evaluating 'window.webkit.messagehandlers')` 1セッション

- サイト側で修正不可、無視で問題なし

### 重大な所見

- **アフィCV 3件**：126万ビュー時点のHANDOFF数値「9件」と整合、**サイト側で全く改善できていない**
