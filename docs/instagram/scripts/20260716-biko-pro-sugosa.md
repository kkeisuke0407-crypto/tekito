## 2. ボディ（プロと素人の差＝プロのすごさ）

1. **距離が「近すぎず、遠すぎず」** — 突っ込めばバレる、離れれば見失う。この絶妙な距離を保ち続けるのがプロの基本技術。
2. **基本は"チーム戦"** — プロは複数名・複数車両で連携することが多い。だから「同じ人がずっと後ろにいる」状態を作らない。素人は一人だから、必ず気づかれます。
3. **「バレそう」を先に察知して、引く** — 対象が振り返る前に離脱。素人は粘って、その一回でバレる。
4. **自分を"風景"にする** — 服装・持ち物・立ち位置を街に溶け込ませ、絶対に凝視しない。
5. **撮るのは「裁判で通用する画」** — ラブホテルの出入りなど、日時と状況が客観的に分かる形で。素人のスマホ写真は"証拠にならない"ことが多い。
6. **そして最大の違い——バレたら、終わり** — 素人が一度失敗して警戒されると、その後プロが入っても証拠が取れなくなります。

**注意（法律）**：バレているのに追い続けると、迷惑防止条例の「つきまとい」や軽犯罪法に問われ、**今度は自分が違法**になることも。プロは危なくなったら即中断します。

**オチ**：「自分でできる」の一線は、あたりをつけるところまで。
決定的な瞬間を、気づかれず、裁判で通用する形で押さえる——これは200年前から"プロの仕事"です。
