## 2. ボディ（構成）

1. **本来は"死罪"だった** — 徳川の世では、不義密通は男女ともに死罪。文字どおり命がけの沙汰でした。
2. **でも、実際は「お金で解決」** — 表沙汰は世間体が悪い。だから内々の示談で済ませるのが一般的でした。
3. **その名も「首代（くびしろ）」** — 首を斬られる代わりに払う金、という意味。江戸での相場は**金7両2分**。
4. **出典はちゃんとある** — 太田南畝『金曽木（かなそぎ）』に「江戸には姦夫の償を金七両二分といふ」。川柳にも「**間男は七両二分と値が決まり**」。
5. **大坂は"5両"** — 同じ不倫でも、土地によって相場が違った。江戸7両2分／大坂5両。
6. **今のお金でいくら？** — 1両をざっくり10万円とすると、**約75万円**（※1両の価値は諸説あり）。

**オチ**：現代の不倫慰謝料の相場は50〜300万円。
——不倫の"値段"は、300年たっても、実はそんなに変わっていません。
ただし今は「**証拠の強さ**」で金額が大きく動く。その順番だけは、間違えないでください。
