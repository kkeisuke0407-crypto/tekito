# 不倫のあと、やり直せる夫婦・別れる夫婦の「分かれ目」

※重要（HANDOFF §12 遵守）: 不倫後の復縁・再構築率の**公的統計は存在しない**（各社アンケートでは再構築5〜7割が多いが結果はバラつく）。数字は断定せず「諸説・アンケートベース」と明記する。

## 1. フック（最初の0.5秒〜2秒）

**不倫のあと、やり直せる夫婦と、別れる夫婦。**
分かれ目は——「**許すかどうか**」ではありませんでした。

---

## 2. ボディ（再構築の"分かれ目"5つ）

1. **した側が、本当に「関係を断った」か** — 中途半端に相手と続いていると、ほぼ再発します。物理的に切れているかが第一条件。
2. **した側に「本気で戻りたい意思」があるか** — された側だけが頑張っても続きません。修復は"二人でやるもの"。
3. **された側が「感情と向き合う時間」を持てたか** — 我慢で蓋をすると、半年後・1年後に爆発しがち。怒りや悲しみを出しきる時間が要ります。
4. **「二度目はない」線引きと"条件"を言葉にしたか** — 曖昧なまま「許す」だけは、再発の温床。ルールを言語化した夫婦は続きやすい。
5. **第三者（カウンセリング等）を挟めたか** — 当事者だけだと感情論で終わる。冷静に話す"場"があると、前に進みやすい。

**補足（正直に）**：不倫後の再構築率に公的な統計はありません。各社のアンケートでは「再構築5〜7割」とされることが多いですが、結果はまちまちです。

**オチ**：やり直すのも、別れるのも、正解はありません。
でも"分かれ目"を知っていれば、**後悔の少ない選択**ができます。一人で抱え込まず、整理する時間を。
