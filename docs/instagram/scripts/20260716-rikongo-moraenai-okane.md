2. **ひとり親医療費助成** — 親子の医療費の自己負担が、ほぼゼロに（自治体制度）
3. **ひとり親控除** — 所得税・住民税が軽くなる（**年35万円**の所得控除）
4. **国民年金・国民健康保険の減免** — 収入が下がったら、申請で保険料が安くなる
5. **母子父子寡婦福祉資金** — 進学・就職の資金を、**無利子〜低利**で借りられる（修学資金など）
6. **高等職業訓練促進給付金** — 資格を取って働くための給付
7. **年金分割** — 婚姻中の厚生年金を分ける。**請求に期限あり**（2026年4月以降の離婚は5年）

**共通点**：どれも役所は"わざわざ教えてくれません"。**知って、自分で申請した人だけ**が受け取れます。

**オチ**：離婚は、終わりじゃなくスタート。
"もらえるはずのお金"を取りこぼさないだけで、生活の足場は大きく変わります。
