## 2. ボディ（構成）

1. **名は、ヴィドック** — 18〜19世紀のフランス人。若い頃は脱獄を繰り返す、筋金入りの犯罪者でした。
2. **犯罪者から、警察のトップへ** — 裏社会を知り尽くした知識を買われ、パリ警察の密偵に。やがて**犯罪捜査局（シュルテ）を創設し、初代局長**にまで上りつめます。
3. **そして"世界初の私立探偵"へ** — **1833年、民間の探偵事務所を創立**。会員は約3,000人。「探偵」という仕事の原型が、ここで生まれました。
4. **日本の探偵の始まりは1895年** — 明治28年、元刑事の**岩井三郎**が東京・日本橋に「岩井三郎事務所」を開設。これが**日本初の私立探偵社**とされています。
5. **共通点は「裏を知る者が、真実を暴く」** — 犯罪者も元刑事も、"隠し方"を知っているからこそ、隠された事実を見つけられた。
