# drafts

借金返済、債務整理、法律事務所口コミ、キーワード調査の下書き置き場です。
ここにあるファイルは基本的に **公開ページではありません**。

## 主なテーマ

| ファイル群 | 内容 |
|---|---|
| `note-karikin-*.md` | 借金返済・返済不能系の記事案 |
| `note-saimuseiri-*.md` | 債務整理とおまとめローン比較の記事案 |
| `note-*-kuchikomi.md` | 法律事務所名 + 口コミ/評判系の記事案 |
| `office-keyword-*.md` | 事務所名・口コミ系キーワード調査 |

## 使い方

- SEO記事やnote記事へ整える前の素材として使います。
- 公開する場合は、事実確認、広告表記、法的表現、ASP規約を確認してから移植します。
